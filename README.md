# 清水河畔保持在线工具


## 安装依赖：

requests==2.18.4

Scrapy==1.5.0

## 使用

- **程序入口文件**： 

  refresher.py

- **运行方法**：源文件运行，在项目目录下执行

  ```
  python refresher.py
  ```

- 运行方法2：下载exe文件直接打开
